

cart.php
	
	contains selected items in shopping cart to be paid using Kaspay.
	
	pay with Kaspay button will be redirected to 
	
	http://kaspay.com/pay/<kode_merchant>/<kode_transaksi>

	kode_merchant  = code created by Kaspay.
	kode_transaksi = code created by merchant which is used to access merchant's API.
	
index.php

	contains sample of simple wsdl created by merchant to be hit by Kaspay to get transaction info
	and to confirm purchase order.
